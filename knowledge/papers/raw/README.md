# Raw Papers

Coloque aqui os PDFs originais.

Este e o diretorio de entrada canonico para ingestao com Docling:

```text
knowledge/papers/raw/
```

Boas praticas:

- nao sobrescrever nomes de arquivo aleatoriamente
- evitar espacos e caracteres exoticos
- usar nomes estaveis
- nao editar o PDF original aqui

Depois de adicionar novos PDFs, rode:

```bash
python scripts/ingest_papers_docling.py
```
