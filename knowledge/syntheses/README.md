# Syntheses

Use este diretorio para documentos de nivel mais alto, por tema.

Exemplos:

- `residual_decoders.md`
- `conditional_priors.md`
- `heteroscedastic_decoders.md`
- `distribution_fidelity_metrics.md`

Regra pratica:

- `notes/` = um paper por arquivo
- `syntheses/` = varios papers consolidados em um tema
